Do I need to fully configure the assistant before I pay?

No. The product is intentionally purchase-first. Sign in, unlock the workspace, then shape the assistant inside the paid flow.

What does Vonza actually use to answer people?

Vonza is still built around the business website as the main source of truth for the website front desk, and the workspace lets you refresh that website knowledge.

Can I add it to the website myself?

Yes. The workspace includes the install handoff and the widget preview so you can place Vonza on the site when you are ready.

What does the client see after payment succeeds?

The paid workspace opens with Overview as the operator home, then Inbox, Calendar, Automations, Customize, and Analytics so the next step is always clear.
